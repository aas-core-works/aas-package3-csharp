    PutPart(Uri, string, byte[])

    Write the content to the package as a package part with contentType.

    Declaration
    public Part PutPart(Uri uri, string contentType, byte[] content)
    Parameters
    Type Name Description
    Uri uri
    string contentType
    byte[] content
    Returns
    Type Description
    Part
    Remarks

    This function needs to be used to put any content into the package.

    You have to introduce the relations by calling, e.g., RelateSupplementaryToSpec(Part, Part).

    The caller needs to be careful not to unintentionally overwrite existing parts which are already related to each other (e.g., do not overwrite a part as a supplementary material which has been made a spec).
